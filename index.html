2º Projeto: Análise de Dados com SQL e Excel(Em andamento)

Desenvolver um projeto pessoal para transformar dados brutos de vendas em insights estratégicos, demonstrando habilidades em ETL (Extract, Transform, Load) e análise de dados.
Processo Implementado: Extração SQL: Criei consulta complexa com múltiplos JOINs entre tabelas de fato e dimensões (FactInternetSales, DimProduct, DimCustomer). Filtrei dados específicos (vendas apenas de 2013). Calculei métricas chave como receita, custo e quantidade vendida
• Em andamento: - Transformação para Excel. Exportar os resultados para o Excel via SQL Server Management Studio. Utilizar o Power Query para tratamento e enriquecimento dos dados. Criar relações entre tabelas para modelo de dados robusto.
✅Microsoft SQL Server (Consulta avançada)
✅Microsoft Excel (Tabelas Dinâmicas e Dashboard)
